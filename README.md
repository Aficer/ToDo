# ToDo
## Приложение ToDo для заметок и дел.
 Учебный проект.
 В этом проекте я использовал и практиковал такие инструменты как:
 1. SwiftUI
 2. UserDefaults
 3. JSONDecoder/Encoder()
 4. Accessibility()
 5. withAnimation
 6. DispatchQueue
 7. UITests/XCTAssert на проверку наличия элемента
